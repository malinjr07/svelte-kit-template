`sv add` updates an existing project with new functionality.

## Usage[](https://svelte.dev/docs/cli/sv-add#Usage)

You can select multiple space-separated add-ons from [the list below](https://svelte.dev/docs/cli/sv-add#Official-add-ons), or you can use the interactive prompt.

## Options[](https://svelte.dev/docs/cli/sv-add#Options)

### \-C, --cwd[](https://svelte.dev/docs/cli/sv-add#Options-C-cwd)

Path to the root of your Svelte(Kit) project.

### \--no-git-check[](https://svelte.dev/docs/cli/sv-add#Options-no-git-check)

Even if some files are dirty, no prompt will be shown

### \--install <package-manager>[](https://svelte.dev/docs/cli/sv-add#Options-install-package-manager)

Installs dependencies with a specified package manager:

-   `npm`
-   `pnpm`
-   `yarn`
-   `bun`
-   `deno`

### \--no-install[](https://svelte.dev/docs/cli/sv-add#Options-no-install)

Prevents installing dependencies
