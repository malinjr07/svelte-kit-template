## Usage[](https://svelte.dev/docs/cli/playwright#Usage)

```csharp

```

## What you get[](https://svelte.dev/docs/cli/playwright#What-you-get)

-   scripts added in your `package.json`
-   a Playwright config file
-   an updated `.gitignore`
-   a demo test
