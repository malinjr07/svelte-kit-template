## Flip.isFlipping

### Flip.isFlipping( target:String | Element ) : Boolean

Returns `true` if the given target is currently being Flipped. Otherwise returns `false`.

#### Parameters

-   #### **target**: String | Element
    
    The target element to check. Can be selector text or an Element
    

### Returns : Boolean[](https://gsap.com/docs/v3/Plugins/Flip/static.isFlipping()/#returns--boolean "Direct link to Returns : Boolean")

Whether or not the given target is currently being Flipped.

### Details[](https://gsap.com/docs/v3/Plugins/Flip/static.isFlipping()/#details "Direct link to Details")

Returns `true` if the given target is currently being Flipped. Otherwise returns `false`.

```less
if (Flip.isFlipping(".box-1")) {  // do stuff}
```
