### .content( element:String | Element ) : Element | self

Gets/Sets the content element.

#### Parameters

-   #### **element**: String | Element
    
    The element that should be used as the content. You can use selector text like `"#content"` or an element reference.
    

### Returns : Element | self[](https://gsap.com/docs/v3/Plugins/ScrollSmoother/content()/#returns--element--self "Direct link to Returns : Element | self")

The content Element (or the ScrollSmoother instance if used as a setter)

### Details[](https://gsap.com/docs/v3/Plugins/ScrollSmoother/content()/#details "Direct link to Details")

Gets/Sets the content element.

### Getter[](https://gsap.com/docs/v3/Plugins/ScrollSmoother/content()/#getter "Direct link to Getter")

```bash
let content = smoother.content();
```

### Setter[](https://gsap.com/docs/v3/Plugins/ScrollSmoother/content()/#setter "Direct link to Setter")

```cpp
// set the content to the element with #content idsmoother.content("#content");
```
