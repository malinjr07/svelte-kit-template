### .wrapper( element:String | Element ) : Element | self

Gets/Sets the wrapper element.

#### Parameters

-   #### **element**: String | Element
    
    The element that should be used as the wrapper. You can use selector text like `"#wrapper"` or an element reference.
    

### Returns : Element | self[](https://gsap.com/docs/v3/Plugins/ScrollSmoother/wrapper()/#returns--element--self "Direct link to Returns : Element | self")

The wrapper Element (or the ScrollSmoother instance if used as a setter)

### Details[](https://gsap.com/docs/v3/Plugins/ScrollSmoother/wrapper()/#details "Direct link to Details")

Gets/Sets the wrapper element.

### Getter[](https://gsap.com/docs/v3/Plugins/ScrollSmoother/wrapper()/#getter "Direct link to Getter")

```bash
let wrapper = smoother.wrapper();
```

### Setter[](https://gsap.com/docs/v3/Plugins/ScrollSmoother/wrapper()/#setter "Direct link to Setter")

```cpp
// set the wrapper to the element with #wrapper idsmoother.wrapper("#wrapper");
```
